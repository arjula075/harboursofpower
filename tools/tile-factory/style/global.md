# Global art direction — overworld tiles

- **Camera:** true top-down aerial (drone / satellite), north-up.
- **Style:** **photorealistic** — like a high-resolution aerial photograph, not painterly game art or flat textures.
- **Tile size:** 512×512 px; content must run **edge-to-edge** with no frame or border.
- **Sea:** natural blue–turquoise grading, ripples, shallow rocky seabed where appropriate.
- **Land:** granular soil, natural shrubs, soft shadows from NW sun.
- **Forbidden:** text, logos, UI, ships (except tiny port overlays), **solid-color margins**, **picture frames**, cartoon style.

See `reference_coast.png` and `reference_style.md`.
